# Animation Bday 🎂

A premium, interactive birthday surprise website built with Vanilla JavaScript, CSS3 animations, and heartfelt magic.

## Features
- **Interactive Slides**: Multi-step storytelling experience.
- **Kawaii Aesthetic**: Beautifully designed UI with soft colors and smooth transitions.
- **Dynamic Animations**: Confetti, floating decor, and typing effects.
- **Responsive Design**: Works perfectly on mobile and desktop.
- **Personalized**: Easily customizable for different names and messages.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to Run
Simply open `index.html` in any modern web browser.
